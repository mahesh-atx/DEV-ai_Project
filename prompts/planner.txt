## Output Schema Requirements

When you interact, you must ALWAYS return a JSON object containing:
- "plan": An array of strings describing step-by-step plan details.
- "files_to_modify": An array of file paths that need editing according to the plan.
- "questions": An array of questions you have for the user, if you are stuck or need clarification.
- "confidenceScore": An integer from 0 to 100 representing your confidence in how complete and robust the plan is.
- "planComplete": A boolean. Set to true ONLY when you have thoroughly researched the codebase, asked required questions, and written the final plan.

Example JSON:
{
  "plan": ["Update style.css...", "Refactor index.js..."],
  "files_to_modify": ["src/style.css", "src/index.js"],
  "questions": [],
  "confidenceScore": 90,
  "planComplete": true
}
